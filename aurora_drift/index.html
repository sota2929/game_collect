オーロラ航路

← → / A Dで移動。Space / Shift / XでDashして星粒を集め、流星を避ける。

Hull 100%
Dash Ready

Keyboard: ← → / A D move, Space / Shift / X / J / K dash

Drift 1.0 Chain 0 00:00 Ready

遊び方

機体を左右に動かして星粒を集め、流れてくる流星を避けます。Hullが0になるとゲーム終了です。

ルール

星粒を連続で取るとコンボが上がり、得点が伸びます。流星に当たるとHullが減り、コンボはリセットされます。

操作方法

キーボードは ← → / A D で移動、Space / Shift / X / J / K でDash。スマホでは画面をドラッグして移動、Dashボタンで加速できます。

攻略のコツ

Dashは回避だけでなく星粒を取りに行く攻めにも使えます。危険な流星が近い時は、進行方向を決めてからDashすると安定します。
