スマホで遊びやすいブラウザゲームを作る工夫

このサイトでは、スマホで片手でも遊べるかどうかをかなり重視しています。具体的には、誤タップしにくいボタンサイズ、画面上部だけで情報が分かるレイアウト、音を出せない環境でも成立する視覚フィードバックを意識しています。

また、通学中や休み時間でも区切りやすいように、1分前後で結果が出る設計を増やしています。短時間で終わるからこそ、「もう一回だけ」が自然に生まれやすく、ブラウザゲームらしい軽さと相性が良いと考えています。
