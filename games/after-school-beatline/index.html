放課後ビートライン

放課後ビートラインは、8レーンに流れるノーツをタイミングよく叩く無料ブラウザリズムゲームです。毎回変わるランダム譜面、5段階の速度レベル、ライフ制で短時間でもしっかり遊び応えがあります。

リズムゲーム音ゲー8レーンライフ制
今すぐ遊ぶ タイミングを見る

遊び方

  • 流れてくるノーツが判定ラインに重なった瞬間に叩きます。
  • 同時押しは最大2つまでです。
  • ミスを3回すると終了です。

ルール

  • Perfect、Great、Goodの判定でスコアが変わります。
  • 譜面は毎回ランダムに生成されます。
  • Level 1〜5でノーツの速さが変わります。

操作方法

  • スマホでは下の8つのボタンをタップします。
  • PCではA / S / D / F / J / K / L / ; キーで操作します。
  • Enterキーでもゲームを開始できます。

攻略のコツ

まずLevel 1〜2で8レーンの位置を覚え、慣れたら速度を上げましょう。ノーツを追いかけすぎず、判定ラインを中心に見るとコンボが安定します。
