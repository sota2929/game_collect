ネオン配達便

← → / A Dで4つのレーンを移動。Space / ShiftでBoost。荷物を拾って障害物を避ける。

Battery 100%
Boost Ready
Speed 1.0 Packages 0 00:00 Ready

遊び方

4つのレーンを切り替えながら荷物を集め、工事バリアやドローンを避けて走り続けます。

ルール

荷物を連続で拾うとコンボが上がります。障害物に当たるとBatteryが減り、0になると配達終了です。

操作方法

PCは ← → / A D で移動、Space / ShiftでBoost。スマホは左右ボタンとBoostボタンで操作します。

攻略のコツ

Boost中は少しだけ接触に強くなります。荷物を取りに行くより、次の障害物のレーンを先に読むとスコアが伸びます。
